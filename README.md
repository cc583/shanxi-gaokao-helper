# 陕西高考志愿填报助手

这是一个本地网页程序，双击 `index.html` 就能使用。

## 当前真实依据

- 用户新增资料 `22-25年全国高校在陕西的院校录取分数.xlsx`，已提取 2025 院校专业组最低分、最低位次，作为当前冲稳保推荐主依据。
- 用户新增资料 `22-25年全国高校在陕西的专业录取分数.xlsx`，已提取 2025 专业最低分、最低位次；输入专业关键词时启用。
- 2026 年陕西省普通高等学校招生工作实施办法政策解读，用于志愿规则提示。
- 陕西招生考试信息网 2026-06-22 “平行志愿”说明，用于确认 2026 年成绩及控制线公布时间提示。
- 2025 年陕西省普通高考一分段统计表，用于按分数估算位次，这是当前程序定位的主依据。
- 陕西省 2025 年普通高校招生录取最低控制分数线，用于批次线差。
- 2024 年陕西省普通高校招生本科一批、本科二批、高职专科正式投档表，仅在没有 2025/2026 匹配结果时兜底参考。
- 用户提供的《2026年历史类在陕招生计划》《2026年物理类在陕招生计划》扫描版 PDF，经本地 OCR 转换为内置招生计划数据。

注意：2025 年陕西已经进入新高考口径，院校录取应优先看“院校专业组”。当前程序已内置 2025 院校专业组录取线；如果你后续有更权威或更新的官方 CSV，仍可导入，程序会优先使用导入数据。

当前已内置 2025 院校专业组录取线 3769 条、2025 专业录取线 12126 条、2026 招生计划约 3.15 万条。2025 录取线用于判断冲稳保，2026 招生计划用于核对今年可报范围。

## 导入数据

点击“下载模板”，按模板填写或粘贴官方 CSV 数据，再通过“导入官方表”上传。推荐字段：

```text
年份,批次,首选科目,院校代码,院校名称,院校专业组,专业,再选科目,计划数,最低分,最低位次,城市,来源
```

导入后，程序会优先使用导入数据参与推荐。

## 重新抓取官方公开数据

如需更新内置数据，运行：

```powershell
python tools/fetch_official_data.py
```

脚本会生成 `data/official-data.js`。
